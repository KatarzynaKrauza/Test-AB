<h1 align="center">Test A/B: Prezentacja produktu a współczynnik konwersji</h1> 

## 📝 Opis projektu 

Celem projektu jest analiza wpływu sposobu prezentacji produktu na współczynnik konwersji w sklepie internetowym z damską biżuterią.  Badanie sprawdza, czy prezentowanie produktów na modelkach (wariant B) zwiększa liczbę zakupów w porównaniu do standardowych zdjęć produktów na białym tle (wariant A).
